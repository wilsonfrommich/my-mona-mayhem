## ✨ ¿Qué es esto?

Mona Mayhem es una **plantilla de taller** que te enseña GitHub Copilot construyendo algo genuinamente divertido: un sitio de arcade retro que compara los gráficos de contribución de GitHub de dos usuarios.

Empiezas desde este repositorio. Copilot hace el trabajo pesado. Tú subes de nivel.

> 🕹️ **Elige tu personaje** — track de VS Code o track de GitHub Copilot CLI

## 🚀 Inicio Rápido

> **Paso 1 — Obtén tu propia copia**

Haz clic en **[Usar esta plantilla](https://github.com/wilsonfrommich/my-mona-mayhem/generate)** para crear un nuevo repositorio, o haz un fork.

> **Paso 2 — Elige tu track y configúrate**

| VS Code | GitHub Copilot CLI |
|---|---|
| Clona tu repo y ábrelo en VS Code | Clona tu repo y abre una terminal |
| Asegúrate de que la extensión GitHub Copilot esté conectada | Instala `copilot` vía `npm install -g @github/copilot` o Homebrew / WinGet |

> **Paso 3 — Empieza a construir**

Abre [workshop/es/00-overview.md](workshop/es/00-overview.md) y sigue la guía.

---

## 🛠️ Requisitos Previos

- GitHub Copilot (Pro, Business o Enterprise)
- Git
- Node.js
- **Solo VS Code:** VS Code v1.107+ con la extensión GitHub Copilot conectada
- **Solo CLI:** Node.js 22+ o Homebrew / WinGet para la instalación nativa
