## ✨ O que é isso?

Mona Mayhem é um **template de workshop** que ensina GitHub Copilot construindo algo genuinamente divertido: um site de arcade retrô que compara os gráficos de contribuição do GitHub de dois usuários.

Você começa por este repositório. O Copilot faz o trabalho pesado. Você sobe de nível.

> 🕹️ **Escolha seu personagem** — track do VS Code ou track do GitHub Copilot CLI

## 🚀 Início Rápido

> **Passo 1 — Obtenha sua própria cópia**

Clique em **[Usar este template](https://github.com/wilsonfrommich/my-mona-mayhem/generate)** para criar um novo repositório, ou faça um fork.

> **Passo 2 — Escolha seu track e configure**

| VS Code | GitHub Copilot CLI |
|---|---|
| Clone seu repo e abra no VS Code | Clone seu repo e abra um terminal |
| Certifique-se de que a extensão GitHub Copilot está conectada | Instale `copilot` via `npm install -g @github/copilot` ou Homebrew / WinGet |

> **Passo 3 — Comece a construir**

Abra [workshop/pt_BR/00-overview.md](workshop/pt_BR/00-overview.md) e siga o guia.

## 🛠️ Pré-requisitos

- GitHub Copilot (Pro, Business ou Enterprise)
- Git
- Node.js
- **Somente VS Code:** VS Code v1.107+ com a extensão GitHub Copilot conectada
- **Somente CLI:** Node.js 22+ ou Homebrew / WinGet para instalação nativa

## ⚙️ Stack Tecnológica

| Camada | Escolha |
|--------|---------|
| Framework | [Astro](https://astro.build/) v5 |
| Runtime | Node.js + adaptador [@astrojs/node](https://docs.astro.build/en/guides/integrations-guide/node/) |
| Fonte | Press Start 2P (jogos retrô) |
| Dados | API de gráfico de contribuições do GitHub |
